# 🎨 OrganicShape Pro | Gerador de Design Orgânico
 
## 📝 Descrição do Projeto
O **OrganicShape Pro** é uma ferramenta avançada de design generativo focada na criação de formas orgânicas fluidas (blobs) para interfaces modernas. O projeto utiliza algoritmos matemáticos para gerar caminhos SVG únicos, permitindo que designers e desenvolvedores criem elementos visuais abstratos com precisão e estilo.
 
Equipado com uma **Inteligência Artificial (Gemini API)** para sugestão de paletas cromáticas e um backend em **Firebase** para persistência de dados, o sistema oferece uma experiência "Cloud-First" onde cada criação pode ser salva, revisitada e gerenciada em uma galeria pessoal segura.

## 🚀 Tecnologias Utilizadas
* **Frontend:** React 19, TypeScript, Tailwind CSS 4.0
* **Animações:** Motion (Framer Motion)
* **IA:** Google Gemini API (via @google/genai)
* **Backend:** Firebase (Firestore & Google Search/Auth)
* **Ícones:** Lucide React
 
## 📊 Resultados e Funcionalidades
O projeto entrega uma ferramenta de alta performance para o fluxo de trabalho criativo:
* **Sincronização em Tempo Real:** Integração total com Firestore para salvamento e recuperação instantânea de designs.
* **Motor Generativo:** Algoritmo dinâmico que ajusta complexidade e contraste de vetores SVG com feedback visual imediato.
* **Curadoria de Cores IA:** Uso de modelos de linguagem para sugerir cores que seguem tendências de design moderno e acessibilidade.
* **Interface Premium:** Implementação de Glassmorphism e AnimatePresence para uma experiência de usuário fluida.

## 🔧 Como Executar
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Configure suas credenciais no arquivo `firebase-applet-config.json`.
4. Inicie o servidor de desenvolvimento: `npm run dev`.
