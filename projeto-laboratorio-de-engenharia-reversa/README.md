## 📝 Descrição do Projeto
O **OrganicShape Pro** (Blob-Studio) é uma plataforma avançada de design vetorial especializada na geração de formas orgânicas (blobs). O projeto originou-se de um processo de **Engenharia Reversa** focado em desconstruir lógicas de interface e algoritmos de geração geométrica, evoluindo para um produto autoral com infraestrutura robusta de nuvem.

Desenvolvido para superar as ferramentas de referência, o sistema aplica conceitos de **Glassmorfismo** em sua interface, oferecendo uma experiência visual premium. A aplicação não apenas gera formas aleatórias, mas permite a personalização técnica de complexidade e contraste, integrando-se ao ecossistema **Firebase** para garantir que cada criação seja única e persistente.

---

## 🚀 Tecnologias Utilizadas
* **Frontend:** HTML5 Semântico + Vanilla JavaScript (ES6+ Modules)
* **Estilização:** CSS3 Avançado (Glassmorphism Architecture & Backdrop Filters)
* **Backend & Auth:** Firebase (Google Authentication & Cloud Firestore)
* **Infraestrutura:** GitHub Pages para hospedagem estática
* **Algoritmos:** Geometria Computacional para renderização de caminhos SVG (Bézier curves)

## 📊 Resultados e Funcionalidades (Diferenciais)
O projeto foi estruturado para transformar a prática de engenharia reversa em inovação competitiva:
* **Persistência em Nuvem (Cloud Firestore):** Salvamento automático dos parâmetros de design (cores, complexidade e caminhos SVG) vinculados ao perfil do usuário.
* **Autenticação Segura:** Sistema de login via Google para personalização e segurança da galeria de designs.
* **Curadoria de Cores via IA:** Algoritmo de sugestão de paletas harmônicas para otimização do fluxo de trabalho do designer.
* **Interface Modernizada:** Aplicação de estilo Glassmorphic com desfoque em tempo real, superando a estética simplista das ferramentas originais.

## 📚 Reflexão Acadêmica

### Questão A: A Formação do Desenvolvedor na Era da IA
A transição da escrita sintática para a descrição lógica exige que o engenheiro júnior atue como um arquiteto de sistemas. Duas competências tornam-se indispensáveis:
1. **Curadoria Técnica:** Capacidade de validar e refinar lógicas geradas por IA para garantir segurança e performance.
2. **Orquestração de Ecossistemas:** Habilidade de integrar serviços (como o Firebase) para criar valor agregado além do código base.

### Questão B: Originalidade vs. Plágio Digital
1. A engenharia reversa assistida por IA torna-se plágio quando há substituição comercial direta sem inovação. O projeto torna-se autoral ao implementar recursos inéditos (Nuvem, IA de Cores, UX Superior).
2. **Diretriz Ética:** Proponho o uso de "Manifestos de Inovação", onde o desenvolvedor documenta explicitamente quais melhorias funcionais foram adicionadas sobre a lógica base.

## 🔧 Como Executar
1. Clone o repositório: `git clone https://github.com/bibijove/Blob-Studio.git`
2. Abra o arquivo `index.html` em um navegador moderno (ou utilize a extensão Live Server).
3. Para funcionalidades de nuvem, certifique-se de configurar suas próprias chaves no objeto `firebaseConfig` dentro do código.
